Science and Technology Foundations

Time-Series Forecasting has been hot research topic since the introduction of machine learning. Dr. Frank Zhu has developed new foundational time-series forecasting theory and mathematical solution. All these leads to the successful machine learning application. Instead of using function based machine learning which current machine learning theory are based on, we invented method based machine learning. Therefore, we can still forecast well without the assumption that the time-series historical data has no pattern

Practice and testing on Stock historical time-series data

Because of the availability of stock historical data, we did extensive testing and practice. with the traditional accuracy verification process. Historical time-series data are divided into training data set and verification data set.

Stock future forecasting performance

  • For most high volatile stocks on regular laptop computing power (in fact, lower volatile stocks are not included in the practice), we have achieved very high accuracy on the verification data set.
  • On real time accuracy performance, if it achieved high accuracy on verification data set, it also achieved high accuracy.
  • We have currently practice our forecasting within 50 days. However, if enough historical data available, it can forecast much beyond that theoretically.

Theory on high accuracy stock forecasting

Our mathematical solution strongly suggests that forecasting accuracy is continuous and consistent. Therefore if it achieves high accuracy on verification data set, it can achieve high accuracy on real time data as well.
